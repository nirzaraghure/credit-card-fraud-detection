# Fraud Detection Model
=======================

## Description
------------

This repository contains a machine learning model for detecting fraudulent transactions. The model is trained on a dataset of historical transactions and uses a random forest algorithm to predict the likelihood of a transaction being fraudulent.

## Tech Stack
------------

* Python 3.x
* Scikit-learn for machine learning
* Pickle for model serialization

## Features
------------

* Trained random forest model for detecting fraudulent transactions
* Model is serialized using pickle for easy deployment
* Requirements file for easy installation of dependencies

## Installation
------------

1. Clone the repository using `git clone https://github.com/your-repo-link`
2. Install the required dependencies by running `pip install -r requirements.txt`
3. Load the model using `import pickle` and `model = pickle.load(open('rf_fraud_model.pkl', 'rb'))`

## Usage Instructions
-------------------

1. Use the loaded model to make predictions on new transactions
2. The model returns a probability of the transaction being fraudulent
